Claude++

Quality-of-life upgrades for the claude.ai interface. Runs entirely in your browser — no account, no backend, nothing ever leaves your device.

Local storage only · No data collected
🎨

Project colors

Assign a color to a project and its chats get tinted sidebar icons, so you can tell at a glance which chat belongs where.

📁

Create project from a chat

Adds “+ Create new project” to the Add-to-project menu, so a loose chat can go straight into a new project in one step.

🛡️

Delete guard

Hardens the project-delete dialog: it names what’s being deleted, warns what’s included, and makes you type the name to confirm.

⏳

Thinking status in the tab

Prefixes the tab title with a status glyph — streaming, finished in the background, or ended with an error — so you know which tab needs you.

💬

Inline asides

Select any passage and ask about it. The answer streams into a card in the margin, anchored to the text — and stays out of your chat history.

⏸️

Draft mode

Press Shift+Tab in the message box to arm Draft mode: type, paste, attach, and dictate freely with no way to send by accident.

Privacy

Claude++ stores everything locally with chrome.storage.local. It requires access to claude.ai only, collects no data, and sends nothing to any server. When you delete a chat or project, anything Claude++ saved for it is deleted too.
